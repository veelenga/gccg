# GitHub Contributions Calendar Games

Classic arcade games with a GitHub contributions calendar twist!

**[Play Now](https://veelenga.github.io/gccg/)**

## Games

### Snake

Collect contribution squares and grow your snake. Higher level cells are worth more points!

![Snake Demo](demo/snake.gif)

### Battle City

Destroy enemy tanks in an arena filled with destructible obstacles.

![Battle City Demo](demo/battle-city.gif)

### Breakout

Break all the contribution blocks with your ball. Different levels require multiple hits!

![Breakout Demo](demo/breakout.gif)

## Development

```bash
npm install
npm run dev
```
